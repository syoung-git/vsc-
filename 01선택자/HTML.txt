-선택자
*		HTML 페이지 내부의 모든 태그 선택 ex) *{color: rea;}
태그		특정 태그 선택 ex) h1{color: rea;}
#아이디명	아이디속성을 가진 태그 선택 ex) #header{ width: 800px }
.클래스명	특정 클래스를 가진 태그 선택 ex) .select { color: red; }
선택자A(부모) (띄어쓰기) 선택자B(자식) ex) #header h1 {color : red;}
